# バッチ処理の状況確認方法

## 現在の状況

バッチ処理はバックグラウンドで実行中です。処理には時間がかかります（CPUモードのため、1ファイルあたり数分〜数十分かかる場合があります）。

## 進行状況の確認方法

### 方法1: 状況確認スクリプトを使用（推奨）
```bash
cd /Users/kayoko.namba/Desktop/transcription
source /Users/kayoko.namba/venvs/whisperx/bin/activate
python check_status.py
```

### 方法2: データベースを直接確認
```bash
cd /Users/kayoko.namba/Desktop/transcription
python db_query.py list
python db_query.py stats
```

### 方法3: ログファイルを確認
```bash
cd /Users/kayoko.namba/Desktop/transcription
tail -f batch_process.log
```

## 処理完了の確認

以下のコマンドで処理が完了したか確認できます：

```bash
python check_status.py
```

すべてのファイルの状態が「completed」になっていれば処理完了です。

## 処理が完了したら

処理が完了したら、以下のコマンドで結果を確認できます：

```bash
# 統計情報を表示
python db_query.py stats

# セグメントを表示
python db_query.py segments

# テキストファイルにエクスポート
python db_query.py export
```

## 注意事項

- 処理はCPUモードで実行されるため、時間がかかります
- バックグラウンドで実行中の場合、プロセスが終了するまで待機してください
- エラーが発生した場合は、ログファイル（`batch_process.log`）を確認してください
