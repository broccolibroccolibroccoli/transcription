# 画面・機能仕様

## 画面構成

### UIデザイン（KARTE サポートサイトテーマ）

[support.karte.io](https://support.karte.io/) のトンマナを参考：
- **アクセントカラー**: ティール `#2aab9f`
- **背景**: 白・薄いグレー（読みやすいサポート・ドキュメント風）
- **カードスタイル**: 角丸12px、ソフトシャドウ
- **ヘッダー**: アクセントカラーの下線
- 設定: `.streamlit/config.toml` および `app.py` 内のカスタムCSS

### 1. ホーム / アップロード画面

| 要素 | 説明 |
|------|------|
| ファイルアップロード | ドラッグ&ドロップまたはファイル選択（mp4, wav, mp3, m4a, flac） |
| 処理オプション | 話者数の指定（min_speakers, max_speakers）※インタビューは2人想定 |
| 実行ボタン | 「文字起こしを開始」 |
| 処理状況 | プログレスバー、ステップ表示（WhisperX → Alignment → Diarization） |

### 2. 処理結果一覧画面

| 要素 | 説明 |
|------|------|
| ファイル一覧 | 登録済みファイルの一覧（既存 db_query list 相当） |
| ステータス | 処理中 / 完了 / エラー |
| 操作 | 詳細表示、要約生成、エクスポート |

### 3. 文字起こし・要約詳細画面

| 要素 | 説明 |
|------|------|
| 文字起こし表示 | 話者別・タイムスタンプ付きで表示 |
| 要約セクション | 「要約を生成」ボタン → LLMで要約 → 表示 |
| エクスポート | テキストファイル、Markdown形式でダウンロード |

## 機能一覧

| 機能 | 優先度 | 既存資産 |
|------|--------|---------|
| 音声ファイルアップロード | 高 | 新規（ファイルパターン拡張） |
| 文字起こし実行 | 高 | batch_process.process_audio_file |
| 話者分離 | 高 | 既存 |
| 文字起こし結果表示 | 高 | db_query |
| 要約生成 | 高 | 新規 |
| 要約結果表示 | 高 | 新規 |
| テキストエクスポート | 中 | db_query.export_to_text |
| ファイル一覧・検索 | 中 | db_query.list_files |
| バッチアップロード（複数ファイル） | 中 | 既存バッチ処理の拡張 |

## 要約の出力形式（案）

```
## 要約

### 概要
（1〜2文で全体の要旨）

### 主なポイント
- ポイント1
- ポイント2
- ポイント3

### 話者別の発言要約
- 話者A: 主な主張・発言内容
- 話者B: 主な主張・発言内容
```

インタビュー向けには「質問と回答の整理」形式も選択可能にする。
