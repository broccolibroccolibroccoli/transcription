# 実行手順まとめ

## 環境情報
- Pythonバージョン: 3.13.3
- Pythonパス: `/opt/homebrew/bin/python3`
- 仮想環境: `/Users/kayoko.namba/venvs/whisperx`

## 実行手順

### 1. 仮想環境のアクティベート
```bash
source /Users/kayoko.namba/venvs/whisperx/bin/activate
```

### 2. 仮想環境がアクティベートされたことを確認
```bash
which python
# 出力例: /Users/kayoko.namba/venvs/whisperx/bin/python
```

### 3. Pythonバージョンの確認
```bash
python --version
```

## 注意事項

### エラーが発生した場合の対処法

1. **Python 3.11が見つからないエラー**
   - エラー: `zsh: no such file or directory: /opt/homebrew/bin/python3.11`
   - 対処: システムにインストールされているPythonを使用する（`python3` または `/opt/homebrew/bin/python3`）

2. **仮想環境のactivateが見つからないエラー**
   - エラー: `source: no such file or directory: /Users/kayoko.namba/venvs/whisperx/bin/activate`
   - 対処: パスを確認し、正しいパスで実行する
   - 確認コマンド: `ls -la /Users/kayoko.namba/venvs/whisperx/bin/activate`

## 推奨される実行方法

### 方法1: 直接sourceコマンドを使用
```bash
source /Users/kayoko.namba/venvs/whisperx/bin/activate
```

### 方法2: 相対パスを使用（プロジェクトディレクトリから）
```bash
cd /Users/kayoko.namba/Desktop/transcription
source ~/venvs/whisperx/bin/activate
```

### 方法3: 仮想環境がアクティベートされているか確認してから実行
```bash
if [ -f "/Users/kayoko.namba/venvs/whisperx/bin/activate" ]; then
    source /Users/kayoko.namba/venvs/whisperx/bin/activate
    echo "仮想環境がアクティベートされました"
else
    echo "エラー: 仮想環境が見つかりません"
fi
```
