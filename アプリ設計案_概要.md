# インタビュー音声 文字起こし・要約アプリ 設計案

## 概要

バッチ処理実行手順（`バッチ処理実行手順.md`）と既存のWhisperX基盤を活用し、**インタビューなどの音声ファイルをアップロードして文字起こし・要約するWebアプリ**を構築する設計案です。

## アプリの目的

| 機能 | 説明 |
|------|------|
| **アップロード** | mp4, wav, mp3, m4a, flac などの音声・動画ファイルをブラウザからアップロード |
| **文字起こし** | WhisperXによる高精度な文字起こし（話者分離対応） |
| **要約** | 文字起こし結果をLLMで要約し、要点を把握しやすくする |

## 既存資産の活用

- `batch_process.py` の `process_audio_file()` ロジック
- `database_schema.py` のDBスキーマ
- `db_query.py` の検索・エクスポート機能
- 仮想環境 `/Users/kayoko.namba/venvs/whisperx`

## 想定ユースケース

1. **インタビュー記録の整理** - 録音したインタビューを文字起こしし、要点を素早く把握
2. **輪読会の記録** - 既存の輪読会ファイルと同様の処理をWebから実行
3. **会議・ミーティングの議事録作成** - 音声をアップロードして要約付き議事録を生成
